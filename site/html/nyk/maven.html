Eclipse:

Convert to maven project: right click on project > configure > convert to maven project

Add dependency: right click on project > maven > add dependency

Reseting project (may help in times of discontent): F5 - refresh Project > clean maven > update project maven > disable dependency management, then maven > enable dependency management close the project then reopen it if you are behind proxy, then configure proxy settings
