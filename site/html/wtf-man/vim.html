VIM

VIM + - open file at line number alt- - escape, key !!!!!!!!!!!!!!!!! ctrl-[ - escape gF - open file under cursor . - execute last command again ; - repeat the last character-wise search help: ctrl ] - follow link ctrl o - go back :q - exit help c - case insensitive search ? - search backward V - linewise visual mode ~ - switch case

  • tab right P - paste before cursor x - delete character :%s/old/new/gc - sed, c means with conformations

MOVEMENT e - end of word E - end of WORD W - start of WORD ge - end of previous word ),( - sentence {,} - paragraph ]],[[ - section

LINES 0 - start of line ^,_ - first non-blank of line +,- - first non-blank of next/previous line Enter - first nonblank of next line

SCREEN LINES g0, g$ - start/end of screen line gm - middle of screen line gk, gj - up/down one screen line

SCREENS H,M,L - go to top/middle/bottom of screen ctrl F,B - page up/down ctrl D,U - half page up/down ctrl E,Y - one more line at bottom/top z Enter,z.,z- - reposition line with cursor at top/middle/bottom

SEARCH ,# - search forward/backward for exact word under cursor g,g# - same, but even when word is embedded % - find match of current brace, quote,... fx,Fx - move cursor forward/backward to x on current line tx,Tx - same, but to one char before x ;/, - repeat/reverse last

LINE NUMBERING numG,:num - goto line number

MARKS '" - move to position of last edit of file . - move to last change in file 0 - position where you last exited vim

INSERT MODE COMMANDS ctrl h - backspace ctrl u - delete line ctrl w - delete previous word

MACROS qx - record characters into x q - stop recording @ - execute x

REGISTERS "ayy - copy line into register a "ap - paste register a :reg - access all registers

MISC

,<< - shift line one tab K - look up word under cursor in man pages

SET COMMAND :set x,nox (also x!),x=value,all,x? - set x/unset x/assign x/print all values/print x command (short) | default | explanation ----------------+-----------+------------------------------------- autoident (ai) | noai | backup (bk) | nobackup | back file up before overwrite ignorecase (ic) | noic | ignore case in search number (nu) | nonu | display line numbers relativenumber | nornu | display relative numbers (rnu) | | shiftwidth= (sw)| 8 | number of spaces added when identing tabstop= (ts) | 8 | tab width wrap | wrap | wrap lines wrapscan (ws) | ws | search wraps around file mouse=a | | mouse mode (use shift when selecting to copy to clipboard)

EDIT COMMANDS [n] operation [m] motion - if both n and m are specified then n x m

c - CHANGE, d - DELETE, y - YANK

C,D,Y - till the end of line cc,dd,yy - current line cf,df,yf - forward up to x

c),d),y) - sentence

g~w - switch case of a word

gu, gU - to lower/upper case

[p - paste but match current indentation r - replace character S - substitute entire line x,X - delete character/delete back . - repeat last change ~ - change case of character ctrl a,x - increment/decrement number under cursor

AUTOMATIC LINEBRAKE: :set tw=80 - set text width gq - formats selected text :set tw=0 - stop formating

COLORSCHEME: :colorscheme darkblue, torte - nice, darker slate, default - less contrast

SPELLCHECK: :set spell spelllang=enus :setlocal spell spelllang=enus :set nospell z= - show suggestions for misspelled word ]s - go to next misspelled word [s - go to previous misspelled word

TABS: :tabe - open new tab gt, gT - go to next/previous tab ctrl + pgup/pgdn - switch tab vim -p - open one tab page per file ZZ - save and close tab (same as :wq)

SPLITS: :vsp - split verticaly ctrl-w, direction - move to split

AUTOCOMPLETE: ctrl+n - show autocomplete suggestions
